# Alien Invasion

> 《外星人入侵》小游戏
>
> 简述：控制飞船，开炮打不断逼近的外星人群
>
> 语言：python3

## Package

- pygame

## 运行

命令：python alien_invasion.py

### 控制

- Left Arrow：向左移动飞船
- Right Arrow：向右移动飞船
- Space：开炮
- q：退出游戏
- e：换大炮弹
- r：换小炮弹
